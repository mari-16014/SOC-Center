Log Source Configuration
Ensure at least one device is configured under:
  • Settings > Log Source Configuration > Devices > Windows Devices
Enable Advanced Audit Policy
  1. Open gpmc.msc (domain) or gpedit.msc (standalone/local).
  2. Edit a GPO applied to Domain Controllers (domain accounts) or target machines (local accounts).
  3. Navigate: Computer Configuration > Policies > Windows Settings > Security Settings > Advanced Audit Policy Configuration > Audit Policies > Account Management.
  4. Open Audit User Account Management – select Success (optional: Failure).
  5. Apply and run gpupdate /force (or wait for normal refresh).
Note
  • Auditing may generate many logs. To manage this:
  • Open Event Viewer → Windows Logs > Security.
  • Increase Maximum log size (at least 30 minutes of data).
  • Select Overwrite events as needed to avoid overflow.
